 ## Run Locally

### Prerequisite
Make sure you have installed these on your local machine.
- Node.js
- MongoDB

Clone the project

```bash
  git clone https://github.com/mihirc0111/Secrets.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon app.js
```
